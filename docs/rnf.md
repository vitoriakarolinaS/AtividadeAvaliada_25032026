# Requisitos Não Funcionais – Sistema FitPass Gym Management

## RNF01 — Disponibilidade
O sistema deve estar disponível 99% do tempo, exceto durante janelas de manutenção programada.

## RNF02 — Segurança
Todos os dados sensíveis devem ser armazenados de forma criptografada.

## RNF03 — Performance
O tempo máximo de resposta para operações simples (ex.: validar acesso) deve ser de até 3 segundos.

## RNF04 — Usabilidade
A interface deve ser intuitiva e responsiva, adequada para desktop e mobile.

## RNF05 — Escalabilidade
O sistema deve ser capaz de suportar múltiplas unidades da rede FitPass.

## RNF06 — Integração
A comunicação com o sistema de catraca deve ocorrer via API REST utilizando JSON.
